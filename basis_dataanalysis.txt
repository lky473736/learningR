Big Data 시대에 하는 분석 : 많은 데이터 -> 학습량 많아짐 -> 오류가 적어짐 -> "AI" 등장

Big Data는 양이 아니라 정보 그 자체를 표현하는 수단

존 매커시 : "인공지능" 용어 대두, Lisp 언어 개발

준비기 : 엘런튜링
요람기 : 존매커시, 마크1 퍼셉트론
제1겨울 : XOR 게이트, 돈 없음, 전문가시스템 등장, prolog
발전기 : MLP 등장, 역전파 알고리즘, 퍼지이론
제2겨울 : 돈 없음
안정기 : 힌튼의 딥러닝 -> DLP 등장, 약인공지능 등장, ReLU 등장
부흥기 : 다양한 인공지능의 등장

약인공지능 : 요즘 대세, 특정 분야에서만 특정 기능 
강인공지능 : 인간 수준의 인지 능력 (AGI)
초인공지능 : 강한 인공지능 1000배

1. 지도학습 : 투입 -> 학습방법을 통한 학습 -> 산출

선형회귀 (Y = aX+b) : 독립변수 -> 종속변수 예측 (독립변수는 기존에 알고 있는 데이터) / 이상치를 제거해야 함
선형회귀에 산점도, 산포도 개념이 쓰임 (상관관계)

의사결정트리, 나이브베이즈, OLS, 선형회귀, 로지스틱회귀, SVM, 앙상블, KNN, CNN(사진), RNN(동영상)
(CNN, RNN은 딥러닝 사용 알고리즘)

2. 비지도학습 : 투입 -> 학습방법 모름 -> 데이터가 갖는 특징, 구조를 스스로 파악 -> 산출

클러스터링, 주성분 분석, K-means, SVD, ICA, Autoencoders (K-means가 중요)

K-means : K 평균을 구해서 군집 분석 (분류)
군집화 : 유사도가 가장 높은 것끼리 묶는 것

3. 강화학습 : 투입 -> 학습 -> 산출 -> 산출데이터의 질에 따라서 학습모델(agent)에게 보상 or 벌 줌
